Contributing to NexChauhan OS

Thank you for your interest in contributing to NexChauhan OS.

We welcome developers, designers, testers, and documentation contributors.

How You Can Contribute

- Report bugs
- Suggest new features
- Improve documentation
- Submit pull requests
- Improve UI/UX
- Help optimize performance

Contribution Guidelines

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Submit a Pull Request.

Please keep your code clean, well-documented, and follow project standards.

Thank you for helping make NexChauhan OS better for everyone. ❤️
