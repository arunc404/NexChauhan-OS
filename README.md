# NexChauhan OS

🚀 Open Source Business Management Platform for Small Businesses.

## About

NexChauhan OS is an open-source platform designed to help small businesses manage their daily operations from a single dashboard.

## Planned Features

- CRM
- Lead Management
- Customer Management
- Billing & Invoicing
- Inventory Management
- Staff Management
- Expense Tracking
- Reports & Analytics
- WhatsApp Integration
- AI Tools
- Multi User Support
- Role Based Access
- Admin Dashboard

## Tech Stack

- PHP
- MySQL
- JavaScript
- Bootstrap

## Roadmap

- Phase 1 - Authentication
- Phase 2 - CRM Module
- Phase 3 - Billing System
- Phase 4 - Inventory
- Phase 5 - AI Assistant
- Phase 6 - SaaS Version
